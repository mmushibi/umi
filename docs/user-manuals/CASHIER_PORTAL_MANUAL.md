## Introduction

The UmiHealth Cashier Portal is a comprehensive point-of-sale system designed specifically for pharmacy operations. This user-friendly interface enables cashiers to process sales efficiently, manage payments, serve customers, and maintain accurate records.

### Key Features
- Fast and intuitive point-of-sale interface
- Multiple payment method support
- Real-time inventory integration
- Customer management
- Receipt generation
- Sales reporting
- Mobile money integration

## Getting Started

### System Requirements
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Stable internet connection
- Valid cashier credentials
- POS hardware (optional: barcode scanner, receipt printer, cash drawer)

### Login Process
1. Navigate to cashier portal URL
2. Enter your username and password
3. Select your branch/shift
4. Confirm starting cash float
5. Click "Start Session"

### Shift Setup
1. Verify cash float amount
2. Check receipt printer status
3. Confirm payment terminal connectivity
4. Review daily targets
5. Start selling session

## Point of Sale (POS)

### Starting a New Sale
1. Click **"New Sale"** or press **Ctrl + N**
2. Add items to cart:
   - **Search**: Type product name or scan barcode
   - **Browse**: Navigate through categories
   - **Quick Add**: Frequently sold items
3. Verify quantities and prices
4. Apply discounts if applicable
5. Proceed to payment

### Adding Items
#### Search Methods
- **Product Name**: Type partial or full name
- **Barcode**: Scan with barcode reader
- **Product Code**: Enter manual product code
- **Category**: Browse by product categories

#### Quantity and Pricing
- Enter quantity manually
- Use + and - buttons for adjustment
- Verify unit price
- Check for promotions
- Apply bulk discounts

### Cart Management
#### Item Actions
- **Edit**: Change quantity or price
- **Remove**: Delete item from cart
- **Hold**: Save sale for later completion
- **Suspend**: Temporarily pause sale
- **Recall**: Retrieve held sale

#### Discount Application
- **Percentage Discount**: Apply percentage off
- **Fixed Amount**: Apply specific amount discount
- **Promotional**: Apply current promotions
- **Customer Discount**: Apply customer-specific discounts
- **Manual Override**: Manager approval required

## Payment Processing

### Payment Methods
#### Cash Payments
1. Enter cash amount received
2. Calculate change due
3. Open cash drawer
4. Give change to customer
5. Confirm payment completion

#### Mobile Money
1. Select mobile money provider
   - MTN Mobile Money
   - Airtel Money
   - Zamtel Money
2. Enter customer phone number
3. Enter payment amount
4. Generate payment request
5. Wait for confirmation
6. Complete transaction

#### Card Payments
1. Select card payment option
2. Enter card amount
3. Insert/swipe card
4. Enter PIN if required
5. Wait for authorization
6. Print card receipt
7. Complete transaction

#### Insurance Payments
1. Verify insurance coverage
2. Calculate patient portion
3. Process insurance claim
4. Apply copayment
5. Generate insurance receipt
6. Document claim details

### Payment Processing Steps
1. Select payment method
2. Enter payment amount
3. Process payment authorization

6. Record transaction details

## Troubleshooting

### Common Issues

#### Sale Processing
- **Issue**: Product not found in system
- **Solution**:
  1. Check product name spelling
  2. Try barcode scan
  3. Search by product code
  4. Check product category
  5. Ask supervisor for assistance

- **Issue**: Payment authorization failed
- **Solution**:
  1. Check payment method details
  2. Verify customer information
  3. Try alternative payment method
  4. Check terminal connectivity
  5. Contact payment provider

#### Hardware Issues
- **Issue**: Barcode scanner not working
- **Solution**:
  1. Check scanner connection
  2. Replace batteries if wireless
  3. Clean scanner lens
  4. Restart scanner
  5. Use manual product entry

- **Issue**: Receipt printer not working
- **Solution**:
  1. Check paper supply
  2. Verify printer connection
  3. Clear paper jam
  4. Restart printer
  5. Use email/SMS receipt

#### System Issues
- **Issue**: System running slow
- **Solution**:
  1. Check internet connection
  2. Clear browser cache
  3. Close other applications
  4. Restart browser
  5. Contact IT support

### Error Messages
- **Out of Stock**: Product not available in inventory
- **Invalid Price**: Price requires manager approval
- **Payment Declined**: Payment method authorization failed
- **Customer Not Found**: No record matching search criteria
- **Network Error**: Connection to server lost

### Emergency Procedures
#### System Failure
1. Use manual sales process
2. Document all transactions
3. Issue manual receipts
4. Contact IT support immediately
5. Enter data when system restored

#### Power Outage
1. Use battery backup if available
2. Complete current transaction
3. Secure cash and valuables
4. Document interrupted sales
5. Follow store emergency procedures

#### Security Incident
1. Remain calm and professional
2. Follow store security protocols
3. Contact store manager
4. Document incident details
5. Report to appropriate authorities
